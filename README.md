# face-detector-using-kinesis-video-streams

This repository contains an AWS SAM application which detects faces using Kinesis Video Streams and Rekognition Video.

For more information, please refer to the following blog post.

https://devnote.tech/2023/12/implementing-face-detector-using-kinesis-video-streams-and-rekognition-video/
